# Chapt App Part 2
### Description
I'm allowing the user to access Quickchat, send messages, show messages sent recently and Quit the Application. The user should enter Firstname, Lastname, Cellphone Number, Username to access the platform
### Running the application
POEApp is running in Netbeans saving messages and menu nevigation
1. Saving a message to JSON
2. Validation Logic before saving
3. Menu System with Switch-Case
4. Overall flow of App  (Login/Register)
                         (Main Menu)
                         (Send Messages)
                         (Save Messages)
                         (Other Option: Coming Soon)
 5. Quit from the application
